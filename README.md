# Student Microservice

This is a simple .NET microservice project with CRUD endpoints for Student, using an in-memory database.

## Features
- CRUD operations for Student
- In-memory database (no external setup required)
- Swagger UI for API testing

## How to Run
1. Restore dependencies:
   ```
   dotnet restore
   ```
2. Build the project:
   ```
   dotnet build
   ```
3. Run the microservice:
   ```
   dotnet run
   ```
4. Access Swagger UI at `http://localhost:5000/swagger` (or the port shown in the terminal)

## Endpoints
- `GET /api/students` - List all students
- `GET /api/students/{id}` - Get student by ID
- `POST /api/students` - Create a new student
- `PUT /api/students/{id}` - Update a student
- `DELETE /api/students/{id}` - Delete a student

## Project Structure
- `Program.cs` - Entry point
- `Startup.cs` - Configuration
- `Student.cs` - Student model
- `StudentContext.cs` - EF Core context
- `StudentsController.cs` - CRUD API controller

## Requirements
- .NET 8 SDK
